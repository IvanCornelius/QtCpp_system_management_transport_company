# История изменений

## [v1] - 21.05.2022
### Добавлено
- Консольный интерфейс.
- Имеет только базовый функционал.
### Примечания
- Для написания исходного кода использовалась кодировка: windows-1251/CP 1251.


## [v2] - 24.11.2022
### Добавлено
 - Вывод списка транспортных средств сортированного по году постановки на учёт.
### Исправлено
- Вместо динамического массива "vector" теперь используется контейнер "map" и "mulitimap".
### Примечания
- Для написания исходного кода использовалась кодировка: windows-1251/CP 1251.


## [v3] - 10.12.2022
### Добавлено
 - Оконный удобный интерфейс.
 - Возможность открывать файл и сохранять базу данных через диалоговое окно. 
 - Больше способов сортировки данных.
### Исправлено
- Разрешения файла базы данных изменено с ".txt" на ".transp". 


## [v4] - 15.12.2022
### Добавлено
 - Диалоговое окно с отображением графика остаточного пробега транспортного средства.
